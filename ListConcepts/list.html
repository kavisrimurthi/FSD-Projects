HTML List Tags Demonstration

Examples of unordered, ordered, and description lists

Unordered List (<ul>)

An unordered list is a list of items where the order does not matter. It is represented with bullet points by default.

  • Item 1
  • Item 2
  • Item 3

Ordered List (<ol>)

An ordered list is a list of items where the order does matter. It is represented with numbers by default.

  1. First item
  2. Second item
  3. Third item

Nested Lists

Lists can be nested inside one another, allowing for hierarchical structuring of content.

  • Item 1
    • Sub-item 1.1
    • Sub-item 1.2
  • Item 2
  • Item 3
    1. Sub-item 3.1
    2. Sub-item 3.2

Description List (<dl>)

A description list (<dl>) is used to display a list of terms and their corresponding descriptions.
